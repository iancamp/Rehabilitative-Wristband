The included libraries (Library folder) must be correctly installed for the application to function. The installation directory will change depending on whether you are using Mac OS or Windows.For Mac OS users:	Drag correct version (32 or 64 bit) of librxtxSerial.jnilib and RXTXcomm.jar into	your /Library/Java/Extensions/ directory.For Windows users:	32 bit: Drag 32 bit rxtxSerial.dll and RXTXcomm.jar into C:\Windows\System32	64 bit: Drag 64 bit rxtxSerial.dll and RXTXcomm.jar into C:\Windows\SysWOW64 	
	If the library is not recognized, make sure this directory exists within your PATH 	environment variable.
